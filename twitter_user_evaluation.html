twitter_user_evaluation.app module¶

This module uses flask to implement a microservices that exposes a REST API for the project’s PHP backend to consume. It takes a couple queries:

GET /?user=user
sends back a json object with analysis of the user
twitter_user_evaluation.app.get_analytics()[source]¶
This method handles a request of the form:
/?user=usertoquery

and returns some analysis on the hashtag.

twitter_user_evaluation.app.not_found(_)[source]¶

This method handles invalid requests by sending a 404 response.
